Class: Phase

jglTask. Phase

new Phase()

This is a phase object, while this constructor is never used, It is useful to see what members a phase object can have.
Properties:
Name Type Description
verbose Number determines if the phase object is verbose with the console.
parameter Object an object to hold parameters for trials.
seglen Array holds definite lengths of all segments within a trial
segmin Array holds the minimum values of all segment lengths
segmax Array holds the maximum values of all segment lengths
segquant Array holds the quantization of all segment lengths
segdur Array a 2-D array, an array for each segment of possible durations
segprob Array a 2-D array, an array for each segment of probabilities for each duration in segdur
segnames Array holds all the segment names
seglenPrecompute Object | Boolean if true, the seglens are precomputed, if false then not, or if a precomputation is provided, then it is validated.
seglenPrecomputeSettings Object an object to determine the precompute settings
getResponse Array a logical array where zero means no response during that segment, one means collect response
numBlocks Number the maximum number of blocks to be run during this phase
numTrials Number the maximum number of trials to be run during this phase
timeInTicks Number 1 means keep track of time in ticks, 0 means don't (WARNING: not tested)
segmentTrace Number the number for the segment trace.
responseTrace Number the number for the response trace.
phaseTrace Number the number for the phase trace.
private Object this is an object with which you may do whatever you like.
randVars Object NOT USED
genRandom Object this is an object whose structure can be found in rand.js
usingScreen Number 1 means yes, 0 means no
html String the name of the html page to be used
